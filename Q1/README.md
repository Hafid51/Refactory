### Soal Pertama - Bahasa Golang

Untuk menjalankan file Golangnya,

```
cd <directoryName>
```

### Contoh

```
cd Q1/
```

## Jika sudah masuk ke directory tempat file tersebut, tinggal eksekusi 
## atau menjalankan programnya saja dengan perintah

```
go run <namaFile>
```

### Contoh

```
go run main.go
```

### Fyi - versi golang 

```
go1.15.8
```
